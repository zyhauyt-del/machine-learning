# 机器学习笔记

整理一些机器学习相关的代码和笔记。

## 机器学习基础算法集

`ml_algorithms/` 目录下是经典算法的手写实现，每个都能独立运行：

- **线性回归** — 最小二乘 + 梯度下降两种实现
- **逻辑回归** — Sigmoid + 交叉熵 + 梯度下降
- **决策树** — CART 分类树（Python手写 + MATLAB实现）
- **随机森林** — Bagging + 随机特征子空间
- **K-Means** — 标准EM迭代 + 轮廓系数评估
- **KNN** — 欧氏/曼哈顿距离，k近邻投票
- **朴素贝叶斯** — 高斯朴素贝叶斯（见 notebook）

每个脚本底部都有简单的测试示例，直接跑就能看效果。
